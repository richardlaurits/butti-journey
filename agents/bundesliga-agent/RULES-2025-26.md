## SCORING SYSTEM

**[Official Bundesliga scoring - statistical analysis based]**

### Base Performance Points
Points awarded for overall match performance based on:
- Passing accuracy
- Touches/involvement
- Defensive actions
- Attacking threat
- Match outcome contribution

**Typical ranges:**
- **Excellent game:** 150-300+ pts
- **Good game:** 80-150 pts
- **Average game:** 40-80 pts
- **Poor game:** 0-40 pts

### Bonus Points Categories

**Goals:**
- Position-dependent multipliers
- Context matters (winning goal, equalizer, etc.)

**Assists:**
- Key passes leading to goals
- Pre-assists sometimes counted

**Defensive Actions:**
- Tackles, interceptions, blocks
- Clearances, recoveries

**Goalkeepers:**
- Saves (especially crucial saves)
- Clean sheets
- Sweeper-keeper actions

**Negative Points:**
- Own goals
- Red/yellow cards
- Penalties missed
- Errors leading to goals
