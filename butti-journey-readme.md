## 🔑 Key Learnings

### What Worked Well
1. **Email-based interfaces** - Interactive buttons for user actions
2. **Strict scheduling** - Reduced frequency to minimize noise
3. **Hybrid approaches** - Combining API data with web scraping
4. **Private/Public separation** - Keeping personal data secure

### What Didn't Work
1. **Over-automation** - Too many notifications created fatigue
2. **Complex scrapers** - Sites with heavy JavaScript often broke
3. **Daily updates** - Too frequent, reduced to weekly where appropriate

### Technical Insights
- **Email parsing** → Most reliable for structured data extraction
- **API-first** → Prefer APIs over scraping when available
- **Gmail IMAP** → More reliable than OAuth for long-term access
- **Cost tracking** → Every operation has a price
