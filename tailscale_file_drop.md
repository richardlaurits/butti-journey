# Tailscale File Drop - How To

## Steg för att skicka filer från iPhone till Linux VM

### 1. Öppna Tailscale på iPhone
- Se till att VPN är på (Connected)

### 2. Hitta min VM
- I enhetslistan, tryck på:
  **richard-laurits-qemu-virtual-machine**

### 3. Skicka filen
- Tryck på **"Select a file to send"** eller **"File Drop"**
- Välj din Apple Health export-fil (export.json)
- Tryck **Send**

### 4. Jag tar emot
- Filen sparas automatiskt på min VM
- Jag analyserar datan
- Skickar rapport till dig!
